## ✨ Key Features

### 🧠 Memory System
- Auto-extracts facts from every conversation
- Stores preferences, goals, projects, patterns, skills, milestones
- Injects memory context into every new conversation
- Commands: `/memory`, `/timeline`, `/clearmemory`

### 🎯 Intelligence Modes
- **Idea Incubator** — cofounder-style idea validation
- **Multiverse Thinking** — same answer from 7 different lenses
- **Thought Companion** — organize messy thoughts
- **Devil's Advocate** — stress-test your decisions
- **Life Coach** — logical + emotional + practical angles
- **Teach Me** (`/teach`) — Socratic learning

### 📊 Weekly Intelligence Report (`/report`)
- Topics explored, ideas captured, projects worked on
- Pattern observed, momentum score, recommended focus

### 🔮 Ambient Intelligence
- Silently detects burnout, flow state, confusion, excitement, frustration
- Adapts tone and depth automatically — never announces it
